# Roadmap

We are building the **full 7-track course before the public launch**, so v1 lands complete.
Lessons are drafted internally track-by-track against a gold-standard quality bar, then
released as one complete drop.

> Live status of every lesson is in [`curriculum.json`](curriculum.json) and on the
> [site roadmap](roadmap.html).

## Milestones

- [x] **M0 — Plan sign-off.** Strategy, curriculum, and tech approach locked. See [`doc/PLAN.md`](doc/PLAN.md).
- [x] **M1 — Skeleton + quality bar.** Static site engine (renderer, nav, catalog, roadmap,
      glossary, widgets, Mermaid), credibility + audience plumbing, and gold-standard sample lessons.
- [x] **M2 — Tracks 0 & 1** (Orientation + Foundations) drafted.
- [x] **M3 — Tracks 2 & 5** (Ecosystems + Governance) drafted.
- [x] **M4 — Tracks 3 & 4** (Operations + Architecture) drafted.
- [x] **M5 — Tracks 6 & 7 + Capstone.** **Full draft content complete — all 92 lessons.**
- [ ] **M6 — Pre-launch hardening.** Human review/refinement of all lessons, accuracy +
      primary-source verification, QA (links, diagrams, mobile, accessibility), SEO,
      README-as-landing polish. *(next)*
- [ ] **M7 — Public launch.** Single complete release of repo + site.
